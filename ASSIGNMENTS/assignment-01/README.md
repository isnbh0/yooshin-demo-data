# 과제 1: 업무 맥락 문서 작성

## 과제 안내

ChatGPT 또는 Gemini를 활용하여 본인의 업무 맥락 문서를 작성합니다.

## 진행 방법

1. [`homework-interviewer-prompt.md`](./homework-interviewer-prompt.md) 파일을 엽니다.
2. 내용 전체를 복사합니다.
3. [ChatGPT](https://chatgpt.com) 또는 [Gemini](https://gemini.google.com)에 새 대화를 열고 붙여넣기합니다.
   - thinking(추론) 기능이 있는 모델을 사용하면 더 깊이 있는 인터뷰가 진행됩니다.
4. AI의 질문에 답하며 인터뷰를 진행합니다.
5. 인터뷰가 끝나면 최종 **업무 맥락 문서**가 생성됩니다.
6. AI에게 **"다운로드 가능한 텍스트 파일로 제공해 주세요"** 라고 요청하여 파일로 저장합니다.
7. 대화 **공유 링크**를 생성합니다. (ChatGPT: 대화 공유 버튼 → 링크 복사 / Gemini: 공유 버튼 → 링크 복사)
