# (주)멋진엔지니어링 — 법인카드 사용내역 배경 설명

## 회사 개요

(주)멋진엔지니어링은 1978년에 설립된 종합 건설엔지니어링 컨설팅 기업이다. 도로·교량·터널·철도 분야의 타당성 조사, 기본/실시설계, 건설사업관리(CM/감리) 및 유지관리 업무를 수행한다. 본사는 서울특별시 영등포구 여의대로(보라매 인근)에 위치하며, 약 1,000명의 임직원이 근무하고 있다.

주요 발주처는 국토교통부, 한국도로공사, 한국철도시설공단, 한국수자원공사, 지방자치단체 등 공공기관이며, 나라장터(G2B)를 통한 공공입찰이 주력 수주 경로이다.

## 총무부 소개

총무부는 사내 시설관리, 사무용품 조달, 경조사 지원, 업무용 차량 운영, 탕비실 관리 등 회사 운영 전반의 행정 지원을 담당하는 부서이다. 5명의 법인카드 사용자가 있다:

| 이름 | 직급/역할 | 주요 지출 |
|---|---|---|
| 최영희 | 팀장 | 고가 사무용품, 외부 접대, 교육비 |
| 강태우 | 차량 담당 | 주유, 하이패스, 세차, 주차 |
| 윤소라 | 용품 담당 | 편의점 간식, 탕비실 물품, 소모품 |
| 박민수 | 신입 | 소액 구매, 간혹 분류 누락 |
| 이정은 | 경조사/행사 | 화환, 배달, 행사 식대, 건물관리 |

## 가맹점명(원본) 비정형 패턴

법인카드 사용내역에서 가맹점명은 카드사 시스템에서 자동으로 전송되는 원본 데이터이다. 같은 가맹점이라도 표기가 제각각인 이유는 카드 단말기(POS/VAN) 등록 정보, 사업자등록증 상호명, 프랜차이즈 본사명 등이 혼재하기 때문이다.

### 주요 패턴

- **법인격 표기 불일치**: `(주)`, `㈜`, `주)`, `주식회사` 가 혼용되거나 아예 생략됨
- **같은 가맹점, 다른 이름**: 예) `오피스디포코리아 보라매` / `(주)오피스디포코리아/보라매` / `오피스디포 BORAMAE`
- **사업자등록 상호명**: 브랜드명 대신 본사명이 표기됨
  - `(주)비지에프리테일` = **CU** (편의점)
  - `㈜지에스리테일` = **GS25** (편의점)
  - `㈜에스케이에너지` / `㈜에스케이네트웍스` = **SK 주유소**
  - `아성산업` = **다이소**
- **POS/VAN 단말기 코드**: `_POS_887`, `_VAN_T301`, `/VAN` 등이 가맹점명 뒤에 붙음
- **플랫폼 접두사**: 배달앱이나 택시앱 결제 시 플랫폼명이 앞에 붙음
  - `배달의민족*원할머니보쌈보라매`
  - `카카오T택시`
  - `쿠팡이츠*본죽보라매`
  - `네이버페이_알파문구`
- **한영 혼용**: `STARBUCKS 보라매R` / `스타벅스코리아_R보라매점`
- **문자열 절단**: 긴 가맹점명이 도중에 잘림 (예: `GS25_보라매역/(주)지에스리테`)
- **고속도로 통행료**: `도로공사하이패스_서울판교`, `고속도로통행료(서울TG-판교)`, `하이패스_서울양재` 등 형식이 다양함
