# yooshin-01

AI 활용 교육 세션을 위한 데이터 패킷. 참가자가 다운로드 후 LLM에 텍스트를 복사/붙여넣기 하며 연습하는 용도.

## 파일 구성

| 파일 | 설명 |
|---|---|
| `법인카드_총무부.tsv` | (주)멋진엔지니어링 총무부 법인카드 사용내역 100건 (2026년 1월) |
| `context.md` | 회사 개요, 부서 소개, 컬럼 설명, 계정과목 정의, 가맹점명 패턴 해설 |
| `problems.md` | 연습 과제 4개 (계정과목 재분류, 가맹점 정규화, 적요 보완, 지출 요약) |

## 사용법

1. 이 폴더를 다운로드
2. `context.md`를 읽고 배경 파악
3. `problems.md`의 과제를 순서대로 수행
4. TSV 데이터를 LLM 대화창에 복사/붙여넣기하여 작업
