# 과제 2: 가맹점 정규화 — 답안

## 분석 접근 방법

`context.md`의 **가맹점명(원본) 비정형 패턴** 및 `problems.md`의 정규화 규칙을 기준으로 100건 전체의 가맹점명을 정규화하였다. 적용한 정규화 규칙은 다음 5가지이다.

1. **법인격 제거**: `(주)`, `㈜`, `주)`, `주식회사` 등 법인격 표기를 모두 제거
2. **POS/VAN 코드 제거**: `_POS_887`, `_POS`, `/POS`, `_VAN`, `/VAN`, `_VAN_T301`, `_T012` 등 단말기 코드 제거
3. **사업자등록 상호 → 브랜드명 변환**: 사업자등록증 상의 본사명을 소비자가 인식하는 브랜드명으로 치환
4. **플랫폼 접두사 분리**: 배달앱/결제앱 플랫폼명을 제거하고 실제 가맹점명만 추출
5. **한영 통일**: 영문 표기를 한글 브랜드명으로 통일

| ㈜지에스리테일 / ㈜지에스25 | GS25 |
| ㈜에스케이에너지 | SK에너지 |

### 플랫폼 접두사 분리 대상

| 플랫폼 접두사 | 분리 방식 | 예시 |
|---|---|---|
| `배달의민족*` | `*` 이후가 실제 가맹점명 | 배달의민족\*한솥도시락보라매 → 한솥도시락 보라매 |
| `쿠팡이츠*` | `*` 이후가 실제 가맹점명 | 쿠팡이츠\*BBQ보라매점 → BBQ 보라매점 |
| `네이버페이_` | `_` 이후가 실제 가맹점명 | 네이버페이\_알파문구 → 알파문구 보라매 |
