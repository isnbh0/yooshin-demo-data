# 과제 1: 계정과목 재분류 — 답안

## 분석 접근 방법

`context.md`의 **계정과목 목록 및 용도**와 **자주 혼동되는 계정과목 경계** 표를 기준으로 100건 전체를 검토하였다. 주요 판단 기준은 다음과 같다.

| 상황 | 흔한 오분류 | 올바른 분류 | 판단 기준 |
|---|---|---|---|
| 스타벅스에서 팀원과 커피 | 접대비 | 복리후생비 또는 회의비 | 사내 직원만이면 접대비 아님 |
| 팀 회식 | 접대비 | 복리후생비 | 외부인 없으면 복리후생비 |
| 하이패스 통행료 | 여비교통비 | 차량유지비 | 업무용 차량이면 차량유지비 |
| 제본/인쇄 | 사무용품비 | 도서인쇄비 | 인쇄·제본 관련은 도서인쇄비 |
| 세차 | 소모품비 | 차량유지비 | 업무용 차량이면 차량유지비 |
| 교보문고 도서 구매 | 사무용품비 | 도서인쇄비 | 도서는 도서인쇄비 |
| 다이소 소모품 | 사무용품비 | 소모품비 | 사무용이 아닌 소모품은 소모품비 |

추가로, 외부 용역(방역 등)이 소모품비로 분류된 건과 문구점에서 "소모품"으로 적요가 기재된 건 등도 주의 깊게 검토하였다.
