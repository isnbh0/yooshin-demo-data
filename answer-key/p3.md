# 과제 3: 적요 보완 — 답안

## 분석 접근 방법

`context.md`의 **부서 구성원 업무 역할** 및 **계정과목 목록**을 기준으로 100건 전체의 적요를 검토하였다. 적요 보완이 필요한 건은 다음 두 유형으로 분류하였다.

### 보완 대상 판단 기준

| 유형 | 정의 | 예시 |
|---|---|---|
| **공란** | 적요 컬럼이 비어 있는 건 | No 8, 10, 27, 28, 51, 68, 97 |
| **불명확** | 적요가 1~2 단어로, 거래의 구체적 목적을 파악하기 어려운 건 | `업무용`, `소모품`, `이동`, `탕비실`, `간식`, `음료`, `식대`, `주유`, `경유`, `세차`, `출장`, `사무용품`, `커피` 등 |

### 보완 시 참조 정보

적요 보완에는 다음 컬럼 정보를 종합적으로 참조하였다.

| 참조 컬럼 | 활용 방식 |
|---|---|
| **가맹점명(원본)** | 거래처 유형 파악 (주유소, 편의점, 음식점, 문구점 등) |
| **계정과목** | 지출 목적 확인 (차량유지비→차량 관련, 복리후생비→직원 복지, 접대비→외부 거래처 등) |
| **사용자** | 담당 업무 맥락 파악 (강태우→차량, 윤소라→용품, 이정은→경조사/행사 등) |
| **합계금액** | 규모 판단 (소액→개인, 대액→부서 단위) |
| **비고** | 추가 맥락 (경조사, 건물관리, 김영란법 한도 이내 등) |

### 보완 원칙

1. 이미 구체적인 적요(예: `신년 탕비실 간식 구매`, `복합기 토너 카트리지 3종`)는 그대로 유지
2. 공란 또는 불명확한 적요만 보완 대상
3. 다른 컬럼으로 추론 가능한 범위 내에서만 구체화하고, 정보가 부족한 경우 `(확인필요)` 표기
4. 동일 유형의 거래에는 일관된 표현을 사용
