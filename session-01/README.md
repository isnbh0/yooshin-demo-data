# Session 01: AI-native로 가는 길

LLM의 세 가지 핵심 원리를 법인카드 데이터로 체험하는 세션입니다.

- **자동완성** — 맥락과 역할이 출력을 결정한다
- **의미공간** — AI는 키워드가 아니라 의미를 이해한다
- **작업기억** — AI는 대화를 넘기면 잊어버린다

## 연습문제 데이터

[`exercises/`](./exercises/) 폴더에 실습용 데이터가 있습니다.

| 파일 | 설명 |
|---|---|
| `법인카드_총무부.tsv` | (주)멋진엔지니어링 총무부 법인카드 사용내역 100건 (2026년 1월) |
| `context.md` | 회사 개요, 부서 소개, 컬럼 설명, 계정과목 정의, 가맹점명 패턴 해설 |
| `problems.md` | 연습 과제 4개 (계정과목 재분류, 가맹점 정규화, 적요 보완, 지출 요약) |

## 사용법

1. `exercises/context.md`를 읽고 배경 파악
2. `exercises/problems.md`의 과제를 순서대로 수행
3. TSV 데이터를 LLM 대화창에 복사/붙여넣기하여 작업
