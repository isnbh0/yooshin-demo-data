# Quadrant Chart (사분면 차트)

사분면 차트는 두 가지 기준축으로 항목의 위치를 표현합니다.

아래 코드를 [mermaid.live](https://mermaid.live)에 붙여넣고 결과를 확인해 보세요.

```
quadrantChart
    title 업무 우선순위 매트릭스
    x-axis "긴급도 낮음" --> "긴급도 높음"
    y-axis "중요도 낮음" --> "중요도 높음"
    quadrant-1 "즉시 처리"
    quadrant-2 "계획 수립"
    quadrant-3 "위임 가능"
    quadrant-4 "보류 가능"
    "월간 보고서 작성": [0.8, 0.7]
    "이메일 정리": [0.3, 0.2]
    "신규 프로젝트 기획": [0.4, 0.9]
    "고객 클레임 대응": [0.9, 0.8]
    "비품 주문": [0.6, 0.3]
    "팀 회식 장소 예약": [0.7, 0.2]
```

## 바꿔 보세요

- 좌표 `[x, y]` 값(0.0~1.0)을 바꿔서 위치를 이동시켜 보세요
- 항목을 추가해 보세요 (예: `연말 감사 준비: [0.5, 0.6]`)
- 축 이름이나 사분면 이름을 바꿔 보세요
