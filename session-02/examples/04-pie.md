# Pie Chart (원형 차트)

원형 차트는 전체 대비 각 항목의 비율을 표현합니다.

아래 코드를 [mermaid.live](https://mermaid.live)에 붙여넣고 결과를 확인해 보세요.

```
pie title 부서별 예산 비율
    "영업부" : 35
    "개발부" : 25
    "인사부" : 15
    "총무부" : 15
    "마케팅부" : 10
```

## 바꿔 보세요

- 숫자를 바꿔서 비율 변화를 확인해 보세요
- 부서를 추가하거나 삭제해 보세요
- `title` 뒤의 텍스트를 바꿔 보세요
