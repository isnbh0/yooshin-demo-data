# Flowchart (순서도)

순서도는 단계별 흐름과 조건 분기를 표현합니다.

아래 코드를 [mermaid.live](https://mermaid.live)에 붙여넣고 결과를 확인해 보세요.

```
flowchart TD
    A[문서 작성] --> B{검토 필요?}
    B -->|예| C[팀장 검토]
    B -->|아니오| D[바로 발송]
    C --> E{승인?}
    E -->|예| F[최종 발송]
    E -->|아니오| G[수정 요청]
    G --> A
```

## 바꿔 보세요

- 결재 단계를 추가해 보세요 (예: `C` 다음에 `부서장 검토` 노드 추가)
- `{검토 필요?}` 같은 조건 분기 안의 텍스트를 바꿔 보세요
- 화살표 위의 라벨 `|예|`, `|아니오|`를 다른 텍스트로 바꿔 보세요
