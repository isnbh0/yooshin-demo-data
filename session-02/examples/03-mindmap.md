# Mind Map (마인드맵)

마인드맵은 중심 주제에서 가지를 뻗어 나가며 관련 항목을 정리합니다.

아래 코드를 [mermaid.live](https://mermaid.live)에 붙여넣고 결과를 확인해 보세요.

```
mindmap
  root((신입사원 온보딩))
    서류
      근로계약서
      보안서약서
      개인정보 동의서
    시스템
      이메일 계정
      그룹웨어
      VPN
    교육
      회사 소개
      직무 교육
      안전 교육
    배정
      좌석
      장비
      멘토
```

## 바꿔 보세요

- 항목을 추가해 보세요 (들여쓰기로 하위 항목 생성)
- 큰 가지를 하나 더 만들어 보세요 (예: `복리후생`)
- 루트 `((신입사원 온보딩))` 텍스트를 바꿔 보세요
