# XY Chart (XY 차트)

XY 차트는 막대와 꺾은선으로 수치 데이터의 추이를 표현합니다.

아래 코드를 [mermaid.live](https://mermaid.live)에 붙여넣고 결과를 확인해 보세요.

```
xychart-beta
    title "월별 매출 추이"
    x-axis ["1월", "2월", "3월", "4월", "5월", "6월"]
    y-axis "매출(억원)" 0 --> 50
    bar [15, 22, 18, 30, 25, 40]
    line [15, 22, 18, 30, 25, 40]
```

## 바꿔 보세요

- 숫자를 바꿔서 차트 모양의 변화를 확인해 보세요
- `bar` 또는 `line` 중 하나를 삭제해 보세요
- x축 라벨과 데이터를 추가해 보세요 (예: `"7월"` 추가 시 데이터 배열에도 값 추가)
